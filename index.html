Отправитель и шапка письма

Текст письма

Отдельный текст для каждого типа визита. Выдели нужное слово или фразу мышью в любом из редакторов ниже, затем нажми «B» или размер — форматируется только выделенный фрагмент.
🔤 Какие плейсхолдеры доступны и как их использовать
Плейсхолдер — это слово в фигурных скобках, которое при отправке письма автоматически заменяется на реальное значение для этого конкретного визита. Работают в тексте письма, в дополнительных страницах и в тексте email-сообщения по умолчанию.
ПлейсхолдерЧто этоПример
Размер: Выдели текст в редакторе, затем нажми нужную кнопку. Шрифт: Aptos
Обычный текст (без выделения) отображается 11pt, не жирным.
Страница 1 — само письмо
Шрифт в PDF — Aptos (с полной поддержкой кириллицы). Жирный и подчёркивание работают в PDF по-настоящему (встроены реальные файлы Aptos Bold). ⚠️ Курсив пока отображается только на экране — файла Aptos Italic ещё нет в PDF-сборке.
Дополнительные страницы (2, 3...)

Текст email-сообщения по умолчанию

Используется только для новых визитов, для которых ты ещё ничего не писал в окне отправки — свой текст для каждого визита сохраняется отдельно и не перезаписывается этим. Общие плейсхолдеры тоже работают: {congregation}, {start_date}, {end_date}.
Тип визита выбирается теми же вкладками выше (Собрание / Группа / Предгруппа).
